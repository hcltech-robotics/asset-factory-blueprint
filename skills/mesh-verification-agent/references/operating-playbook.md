# Operating playbook

Use mesh-verification-agent immediately after reconstruction or source conditioning.

Resolve the candidate, run deterministic diagnostics, generate the fixed review bundle and call the configured vision reviewer. Apply only registered local repairs. Execute governed reconstruction resubmissions when regeneration is required. Re-run the complete verifier after every candidate change.

Return validated only when the approval and promotion records match the current candidate checksum. Otherwise return blocked or review required and keep every downstream geometry deliverable unavailable.
